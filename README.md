# VoiceVoxChatReader_for_FF14

VOICEVOXを利用してFF14のチャットを読み上げるACTプラグインです。

※VOICEVOXが日本語向けのTTSエンジンのためUI内では「VOICEVOX チャット読み上げ for FF14」という表記にしています。

プレイヤー（およびNPC）ごとの読み上げカスタマイズが可能です。名前で識別されます。

## 主な機能

* **VOICEVOXエンジン管理**: プラグインからVOICEVOXエンジンをバックグラウンドで起動・管理します。個別の起動は不要です。
* **プレイヤー個別設定**: プレイヤーごとにボイス、読み上げ速度、ピッチ、読み上げ名を個別に設定できます。
* **読み上げ辞書**: 単語の読み方のカスタマイズができますが、簡易的なもので一致したものはすべて置き換える仕様です。
* **チャンネル選択**: 読み上げるチャットチャンネルを詳細に選択可能です。
* **スパム回避機能**: 短時間の連続投稿を検知すると読み上げをスキップします。（エモート連打など）

## 導入方法

1. 本プラグインをダウンロードし、ACTのプラグインフォルダにファイルを配置します。
2. ACTの「Plugins」タブから本プラグインの `.dll` ファイルを選択して読み込みます。
3. 設定画面の「VOICEVOX接続設定」にて、VOICEVOXのインストールパス（`VOICEVOX.exe`）を指定します。

## 使い方

* **オプション**: 「VOICEVOX接続・起動設定」からパスを確認し、VOICEVOXエンジンの起動・接続を行います。動作が不安定な場合は「エンジン再起動」ボタンでプロセスをリセットできます。
* **プレイヤー設定**: チャットを検知したプレイヤーは自動的にドロップダウンリストへ追加されます。「プレイヤー設定」タブから、好みのボイスや個別の読み上げ名を登録することが可能です。
* **辞書設定**: 特定の単語や略語の読み方を変更したい場合は「辞書設定」タブで行えます。VOICEVOXの辞書とは別ですが、プラグインで変換された単語がVOICEVOXに渡されます。
* **デバッグログ**: 読み上げが上手くいかない場合は、チャンネル設定の下にある「デバッグログを表示する」をオンにしてログを確認してください。

## 注意事項

* 本プラグインの利用には、ACT本体およびFFXIV ACT Plugin、[VOICEVOX](https://voicevox.hiroshiba.jp/)のインストールが必要です。
* チャットであっても「定型文」は読み上げ対象外です。
* 英数字の大文字・小文字は区別して扱われます。
