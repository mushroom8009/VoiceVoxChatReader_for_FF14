# VoiceVoxChatReader_for_FF14
VOICEVOXを利用してFF14のチャットを読み上げるACTのプラグインです</br>
ACTとVOICEVOXが必要です</br>
ACTのプラグインフォルダにダウンロードしたZIPファイルの中身を入れて、他のプラグイン同様、ACTで読み込んでください</br>
VOICEVOXのエンジンを利用するので、VOICEVOXを起動していなくても読み上げます</br>
（裏でエンジンを起動しているので負荷はそんなに変わらないです）</br>
</br>
ACTが動かないと当然動きません</br>
</br>
チャット読み上げるのにいろんなソフト起動するのがめんどくさくなったので、ついでに人によって音声も変えられたらいいなとおもい作りました</br>
リリース時点(ｖ0.9.7)で細かなテストは一切行っていませんのでご了承ください</br>
</br>
いままで読み上げ系のソフトを使っていた方ならある程度は見ればわかるようになってるとおもいます</br>
</br>
</br>
辞書はVOICEVOXの辞書から独立しています（プラグイン独自の辞書）</br>
</br>
ACT＞本プラグイン(辞書にマッチした文字列を置き換えてVOICEVOXにわたす)＞プラグインから受け取った文字列をVOICEVOXが読み上げ</br>
という流れなですが、管理が面倒だとおもうのでどちらかに統一したほうが運用は楽だとおもいます</br>
</br>
まだ細かく作り込んでいないので、例えば英字の場合、大文字小文字、全角半角を別のものとして扱います</br>
いずれは正規表現などにも対応しようかと思います</br>
